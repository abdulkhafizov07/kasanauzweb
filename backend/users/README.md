# User (Foydalanuvchilar quyi tizimi)
<img src="../.anatomy.png" alt="Anatomy" />

### 1. Vazifalari:
  - Foydalanuvchilarni ro‘yxatdan o‘tkazish va tizimga kirish imkoniyatini taqdim etish.
  - Foydalanuvchi profillarini boshqarish va ularning ma’lumotlarini yangilash imkonini yaratish.
  - Kirish huquqlarini boshqarish va foydalanuvchilarni rollarga ajratish (kasanachi, ish beruvchi, administrator, moderator).

### 2. Asosiy xususiyatlari:
  - Xavfsizlikni ta’minlash (login, parol).
  - Tizimning boshqa quyi tizimlari bilan uzviy bog‘langanligi.
  - Shaxsiy ma’lumotlarni himoya qilish (mavjud qonunlarga mos ravishda).

### 3. Ierarxiya darajalari:
  - Adminstrator
  - Moderator.
  - Foydalanuvchilar (kasanachi va buyurtmachi).

### 4. Markazlashtirish darajasi:
  - Foydalanuvchi ma’lumotlari yagona bazada saqlanadi va barcha quyi tizimlar bilan integratsiya qilinadi.
